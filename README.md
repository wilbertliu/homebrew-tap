# Wilbertliu Tap

## How do I install these formulae?

`brew install wilbertliu/tap/<formula>`

Or `brew tap wilbertliu/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "wilbertliu/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
